Personal blog website migration

Last updated on Saturday, February 11, 2023 4:08 PM

I registered GitHub pages with Gmail liwenda.edu@gmail.com a few days ago. But it has some problems recently, so I decide to migrate my blog website address. The new website address is liwenda-yanjin.

Updated on 2023.02.11

I successfully verified Gmail liwenda.edu@gmail.com with virtual mobile phone number, So the GitHub Pages website is still liwendaucas.

 

本博客网站我是使用11月份初刚申请的谷歌邮箱注册了Github账号并开通了GitHub Pages，但是最近我发现我注册GitHub使用的gmail邮箱 liwenda.edu@gmail.com 出现了手机号码无法验证的问题，使用别人的手机号也不行，导致我现在无法在新设备上登录我的GitHub账户（需要邮箱验证，但邮箱已无法进入），故我会重新申请一个新的GitHub Pages，新的个人博客网站域名为liwenda-public。

以后再注册gmail时，最好再添加一个辅助邮箱，以防意外情况。

2022.11.25日更新

今天登录谷歌邮箱，发现 liwenda.public@gmail.com 邮箱也挂了，又得手机验证，我干脆用outlook邮箱创建了一个新的GitHub账户，域名更新为liwenda-yanjin。

2023.02.11日更新

今天闲来无事，刷B站时看到ChatGPT的好几个热门视频，突然想到自己去年12月初注册ChatGPT时用的接受手机号验证码的事情（虚拟电话号码网站），想着用国外的手机号码来解锁之前被封的几个谷歌邮箱，包括这个GitHub Pages所用的谷歌账户，选了俄罗斯和印度尼西亚的几个电话号码，大部分成功验证通过，但这个Github Pages所使用的邮箱 liwenda.edu@gmail.com 是被谷歌封了，我已经申诉了，估计过几天应该可以用。那么网站迁移工作我暂时不想动了，liwendaucas这个域名就先用着吧，如果以后换电脑重新配置hexo、git的环境或者以后毕业了想重新开一个个人网站，到时候可以用回我的liwenda-public域名。

对了，验证手机号码后我翻了一下 liwenda.public@gmail.com 邮箱，发现自己在2022.11.25日把 liwenda-public 的GitHub Account删除了，90天内是没办法再登录的，我说我怎么想重新用 liwenda.public@gmail.com 邮箱登录GitHub不管用，找回密码也没有绑定的邮箱，那只能90天后再把这个账户给重新注册一下，以后有机会的话再迁移。

我发现，可能是我原来的手机号码注册的谷歌邮箱太多了，注册一个，没几天就又得手机验证，添加辅助邮箱也不管用，不过还好有虚拟电话号码网站，liwenda-yanjin 这个域名也是以后有机会再用吧。
